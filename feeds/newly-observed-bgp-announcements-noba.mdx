---
title: "Newly Observed BGP Announcements (NOBA)"
description: "New or updated BGP announcements can reveal important network shifts and early indicators of malicious activity."
---

<Info>
  An announcement is considered newly observed if any of its attributes change,
  or if the announcement has not been observed for more than 30 days.
</Info>

## Format

```json JSON
{
  "timestamp": "2024-12-30T04:12:57Z",
  "prefix": "130.137.28.0/24",
  "origin_asn": 16509,
  "as_path": [40994, 16509],
  "communities": [
    { "asn": 17152, "value": 1 },
    { "asn": 40994, "value": 1 },
    { "asn": 40994, "value": 64203 }
  ],
  "peer": {
    "ip": "193.203.0.75",
    "asn": 40994
  }
}
```

## Download

Newline-delimited JSON dumps of the previous day's stream are made available daily at the following URL.

```
https://feeds.opensystems.sh/noba
```

<Note>
  The file is updated every day between midnight and 1am UTC. Use conditional
  requests (`If-Modified-Since`) or make use of `HEAD` requests to avoid
  unnecessary bandwidth usage.
</Note>
