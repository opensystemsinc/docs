---
title: "Newly Observed Hostnames (NOH)"
description: "NOH focuses on the first time a hostname is observed in the DNS and can help uncover malicious activity at the subdomain level."
---

## Format

```json JSON
{
  "timestamp": "2022-02-14T18:25:43.511Z",
  "hostname": "login-paypal.ddns.net"
}
```

## Rate

The current rate is around **15,000,000** newly observed hostnames per day.

## Download

Newline-delimited JSON dumps of the previous day's stream are made available daily at the following URL.

```
https://feeds.opensystems.sh/noh
```

<Note>
  The file is updated every day between midnight and 1am UTC. Use conditional
  requests (`If-Modified-Since`) or make use of `HEAD` requests to avoid
  unnecessary bandwidth usage.
</Note>
